# Classic Tailor's - Premium Indian Tailoring Website

A beautiful, responsive website for **Classic Tailor's** вЂ” a traditional Indian tailoring shop located in Mirganj, Gopalganj, Bihar, owned by **Masoom Ahmad Siddique**.

## Features

- **Responsive Design** - Works perfectly on mobile, tablet, laptop, and desktop
- **Modern UI/UX** - Clean, elegant design with smooth animations
- **Landing Page** - Hero section with shop branding
- **Services Section** - Display all tailoring services with pricing
- **Gallery** - Portfolio showcase with lightbox
- **Testimonials** - Customer reviews slider
- **Contact Form** - Appointment booking form
- **WhatsApp Integration** - Direct chat button
- **Scroll to Top** - Easy navigation
- **SEO Optimized** - Meta tags and structured data

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: SCSS (Sass)
- **Icons**: Lucide React
- **Animations**: CSS Animations + Intersection Observer

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/sfaiz801/classic-tailors.git

# Navigate to project directory
cd classic-tailors

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

### Deployment

The project is configured for static export. After building, the `dist` folder contains the static files ready for deployment.
